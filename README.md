## Quick Start

### Web App
1. Open `portfolio_tracker.html` in any modern web browser
2. Add your investments using the form
3. Data saves automatically in your browser

### Excel Spreadsheet
1. Open `portfolio_tracker.xlsx` in Excel or Google Sheets
2. Update yellow-highlighted cells with current prices
3. All calculations update automatically

## Usage

### Adding Investments
Enter the following for each asset:
- Asset Name (e.g., "Apple Inc.")
- Ticker Symbol (e.g., "AAPL")
- Quantity owned
- Purchase price per unit
- Current price per unit

### Updating Prices
**Excel**: Update the yellow-highlighted cells in column E
**Web App**: Delete and re-add the asset with new current price
